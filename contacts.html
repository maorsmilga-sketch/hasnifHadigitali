🔧 הגדרת Firebase — פעם אחת בלבד

אחרי ההגדרה, מאור ועידו יוכלו לראות ולעדכן נתונים בזמן אמת מכל מכשיר.

1
פתח את Firebase Consoleלך ל console.firebase.google.com ← Add project ← שם (למשל poker-contacts) ← בטל Analytics ← Create project
2
הפעל Realtime Databaseמהתפריט: Build → Realtime Database → Create Database ← אזור ← Start in test mode → Enable
3
קבל Configדף הבית ← לחץ </> ← הזן שם ← Register app. תקבל:
const firebaseConfig = { apiKey: "AIzaSy...", authDomain: "poker-contacts.firebaseapp.com", databaseURL: "https://poker-contacts-default-rtdb.firebaseio.com", projectId: "poker-contacts", storageBucket: "poker-contacts.appspot.com", messagingSenderId: "123456789", appId: "1:123:web:abc123" };
4
הדבק ב-HTMLפתח הקובץ בעורך טקסט ← חפש PASTE_YOUR_API_KEY ← החלף את כל 7 השדות ← שמור ← שלח לעידו ולמאור.
ⓘ בחינם לצמיתות — תוכנית Spark של Firebase, אין צורך בכרטיס אשראי.
